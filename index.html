Cribbage for iMessage Privacy Policy

1. Introduction

This page describes the privacy policy for the Cribbage for iMessage app. It will be kept up to date with the latest information regarding the app policy.

2. Information We Collect

The Cribbage for iMessage app does not collect any information beyond the default crash report statistics automatically collected by iOS.

3. Contact Us

To get in contact regarding data privacy or any other questions or issue with the app, file an issue at the [Cribbage for iMessage GitHub page](https://github.com/codepoet509/CribbageMessageAppInfo/issues).
